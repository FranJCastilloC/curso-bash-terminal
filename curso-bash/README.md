# Curso de Bash y Terminal

Aplicación web interactiva que presenta un curso completo de Bash y la terminal de Linux, organizado en módulos, secciones y fases de proyecto.

Construida con **React 19**, **Vite** y **Tailwind CSS 4**.

## Desarrollo

```bash
npm install
npm run dev
```

## Build de producción

```bash
npm run build
npm run preview
```

El contenido del curso vive en [`src/bash_terminal_course.jsx`](src/bash_terminal_course.jsx) como estructura de datos (secciones, módulos, comandos, ejemplos y fases de proyecto). Un resumen textual adicional está disponible en [`Contenido.md`](Contenido.md).
